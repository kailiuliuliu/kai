01 NOTE hello hugo

Jan 29, 2019 13:58 · 382 words · 1 minute read

一、 环境

        time: 2019.01.29
          OS: Ubuntu 18.04.1 LTS
     OS type: 64-bit
hugo version: v0.53

二、 操作，在terminal中

  1. hugo version //版本v0.53

  2. cd /media/fengkai/BLACK/programming/08_wd_html

  3. hugo new site hugowebsitekai //新建一个项目

  4. cd hugowebsitekai/themes/ //到themes文件夹下，去下载hugo主题

  5. git clone https://github.com/budparr/gohugo-theme-ananke.git

  6. cd .. //到项目目录下

  7. //配置项目目录下的config.toml文件

  8. hugo new posts/my-first-post.md //新建一个markdown文章，会新建content/post和resources文件夹

  9. //手动编辑markdown文档，设置draft: false

  10. hugo server //启动服务器，才能看见主题效果

  11. //浏览器打开http://localhost:1313 查看效果，终端ctrl+c停止服务器

  12. hugo //编译，输出为public文件夹（可以在config.toml里设置输出路径）

三、 附注

  • markdown文档，引用的图片放在static里的文件夹里，比如static/images/img.jpg

  • 去主题项目里，css文件中，设置字体大小

  • 参考资源来自网络
