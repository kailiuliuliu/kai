三、 tomcat 局域网访问

windows下 允许java通过防火墙  不用动路由器

二、 为eclipse添加tomcat的插件

1.下载tomcat对eclipse的插件。
  下载地址：http://www.eclipsetotale.com/tomcatPlugin.html

2.下载后，解压。解压后的文件放到eclipse的plug目录中。

3.如果没有出现这三个图标，那么选择'Window>Customize Perspective...>Commands'，并在'Available command groups'中勾选'Tomcat'

4.还需要配置tomcat的版本，和tomcat的home，Window>perference

一、 安装java开发环境

1.安装java   jdk即可  
    配置环境变量 %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
2.解压缩eclipse 
3.解压缩tomcat （不是必要配置环境变量） 
    在eclipse中创建server时选择tomcat的路径  注意server的deploy目录
4.解压缩mysql 
    配置环境变量D:\programming\mysql\mysql-5.6.24-win32\bin 
    修改my-default.ini文件（或者建立my.ini文件 必英文编码（如windows中的ANSI），不能是UTF-8或GBK等） 
    添加
    ###
    [mysqld] 
    basedir=D:\programming\mysql\mysql-5.6.24-win32
    datadir=D:\programming\mysql\mysql-5.6.24-win32\data
    ###
    管理员cmd //mysqld -install //mysqld -remove  
    修改 计算机管理 服务里的mysql 手动启动
    管理员cmd //net start mysql //net stop mysql
    cmd>    mysql -uroot -p
        show databases;
        grant select,insert,update,delete on *.* to user1@localhost Identified by "user1";

    解压缩SQLyog 8.32
